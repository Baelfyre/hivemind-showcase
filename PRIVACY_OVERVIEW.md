# Privacy Overview

HiveMind is designed around a privacy-aware collaboration model. The product
separates public information, workspace activity, administrative visibility, and
protected service responsibilities.

## User Control

The workspace experience is designed to keep the user in control of goals,
context, and decisions during AI-assisted work.

## Data Posture

HiveMind favors a local-first user experience where practical, with protected
service validation for sensitive actions.

## Showcase Limits

This showcase does not include private user data, production datasets,
operational logs, service internals, or protected configuration.

## Public Materials

Public materials in this repository are limited to high-level product
documentation, ownership notices, and public brand assets.
